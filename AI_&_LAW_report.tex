\section{What are the pros and cons of using this exoskeleton?}
+ more efficiency 
+ less injury prone / work can appear as more safe
+ equality -> each worker has same strength -> standardize performance
+ less turnover -> older employees can still work
+ better company image

- requires skill -> license/training
- needs data -> privacy risks
- more expensive -> maintenance 
-- needs experts to handle and fine-tune (it manager -> exoskeleton manager)
- discrimination (ex: heart condition)
- stressful / unconfortable / limits movement
- cultural resistance


\section{It is possible to impose the use of these systems?}
Both:
- medical checkup
- if a debilitating medical condition arises after , the employee will be moved to another task (no firing -> demotion/demansionamento)

Asking a "normal employee" to use the exoskeleton (the job application did not mention exoskeletons)
- any device that alters radically the workflow not mentioned in the contract signed from the employee must be negotiated: the employee must consent
- If the employee provided consent they must be adequately trained and its pay grade could be adjusted


Specifically hiring someone to work with an exoskeleton (the job application specifically mentions working with an exoskeleton) 
- it can be required at the moment of hiring(just like you can require someone with a drivers license)
- medical checkup must be passed in order to be hired
